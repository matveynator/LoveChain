# LoveChain iOS

This folder is reserved for the future iPhone version of LoveChain.

Planned responsibilities:

* Swift or SwiftUI application;
* local LoveChain storage;
* device signing with Apple security APIs;
* import and export compatible with the Android JSON format;
* future LoveMap background-location implementation with clear mutual consent.

The iOS app must stay compatible with the LoveBlock format used by `ANDROID/`.
